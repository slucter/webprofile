# Handoff Template

Salin blok di bawah dan isi setiap bagian sebelum menyerahkan kendali ke agent berikutnya. Handoff tidak lengkap = fase belum selesai.

```
Dari: <agent>
Ke: <agent>
Tanggal: <YYYY-MM-DD>
Iterasi: <n>

## Ringkasan kerja
<apa yang dikerjakan dan hasilnya, 2-5 kalimat>

## Artefak yang diubah
- <path/file> — <perubahan singkat>

## Yang perlu diperhatikan agent berikut
- <asumsi, jebakan, keputusan terbuka, hal yang sengaja ditunda>

## Status DoD
- [ ] <kriteria 1 dari persona> — terpenuhi / tidak / N/A
- [ ] <kriteria 2>
Kesimpulan: LOLOS / TIDAK LOLOS — <alasan bila tidak lolos>
```
